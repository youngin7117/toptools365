진법 변환기 사용 방법

입력 진법을 선택하고 숫자를 입력하면 2·8·10·16진수로 즉시 변환됩니다. 결과 박스를 클릭하면 복사됩니다.

진법이란?

2진수 (Binary): 0과 1만 사용. 컴퓨터가 데이터를 처리하는 기본 방식.
8진수 (Octal): 0~7 사용. Unix 파일 권한(chmod) 등에서 활용.
10진수 (Decimal): 0~9 사용. 일상에서 쓰는 숫자 체계.
16진수 (Hexadecimal): 0~9와 A~F 사용. 색상 코드(#FF5733), 메모리 주소 등에서 사용.

자주 묻는 질문

16진수에서 A~F는 무엇인가요?
A=10, B=11, C=12, D=13, E=14, F=15를 의미합니다. 10진수로 10 이상의 값을 한 자리로 표현하기 위해 사용합니다.
색상 코드 #FF5733은 어떻게 해석하나요?
#FF5733에서 FF=빨강(255), 57=초록(87), 33=파랑(51)입니다. 각 2자리 16진수가 0~255의 색상 채널 값을 나타냅니다.
비트 AND 연산이란?
두 숫자의 각 비트를 비교해 둘 다 1일 때만 1을 반환합니다. 예) 1100 AND 1010 = 1000
2진수로 변환하는 방법은?
10진수를 2로 나눈 나머지를 아래에서 위로 읽으면 됩니다. 예) 13 ÷ 2 = 6...1, 6 ÷ 2 = 3...0, 3 ÷ 2 = 1...1, 1 ÷ 2 = 0...1 → 1101
8진수는 어디에 사용되나요?
Unix/Linux 파일 권한(chmod)에서 8진수를 사용합니다. 예) chmod 755 = rwxr-xr-x, 7=rwx, 5=r-x입니다.
255를 16진수로 바꾸면?
255(10진수) = FF(16진수)입니다. 웹 색상에서 흰색(#FFFFFF)의 각 채널은 255, 즉 FF로 표현됩니다.
비트 OR 연산이란?
두 숫자의 각 비트를 비교해 하나라도 1이면 1을 반환합니다. 예) 1100 OR 1010 = 1110
1024를 2진수로 변환하면?
1024(10진수) = 10000000000(2진수)입니다. 2의 10승이므로 1 다음에 0이 10개 붙습니다.
