텍스트 비교기

두 텍스트의 차이점을 줄 단위·단어 단위로 즉시 비교합니다. 추가·삭제·변경 항목을 색상으로 구분합니다.

텍스트 비교기 사용 방법

왼쪽(A)에 원본 텍스트, 오른쪽(B)에 비교할 텍스트를 입력하고 비교하기 버튼을 누르면 차이점을 즉시 확인할 수 있습니다.

주요 기능

줄 단위 비교: 줄 전체 기준으로 추가·삭제·변경 표시
단어 단위 비교: 변경된 단어만 인라인 강조 표시
통합 보기: 변경 사항을 하나의 화면에서 확인
나란히 보기: A·B 텍스트를 좌우로 나란히 비교
대소문자·공백 무시: 옵션으로 세밀한 비교 가능

이런 경우 유용해요

계약서·문서 수정 전후 비교, 코드 변경사항 확인, 번역문 원문과 비교, 작문 초안과 수정본 비교 등에 활용할 수 있습니다.

자주 묻는 질문

줄 단위와 단어 단위 비교의 차이는?
줄 단위는 줄 전체를 기준으로 추가·삭제·변경을 표시합니다. 단어 단위는 변경된 단어만 강조해 더 세밀하게 비교할 수 있습니다.
입력한 텍스트가 서버에 저장되나요?
아니요. 이 도구는 브라우저에서만 동작하며 입력한 텍스트는 서버에 전송·저장되지 않습니다.
텍스트 길이 제한이 있나요?
별도 제한은 없지만 매우 긴 텍스트(수만 줄 이상)는 처리 속도가 느려질 수 있습니다.
대소문자를 무시하고 비교할 수 있나요?
네, 상단 옵션에서 '대소문자 무시'를 체크하면 대소문자 구분 없이 비교합니다.
코드 비교에도 사용할 수 있나요?
네, 코드도 텍스트이므로 비교할 수 있습니다. 줄 단위 비교가 코드 변경사항 확인에 적합합니다.
