마크다운 변환기

마크다운을 입력하면 실시간으로 HTML 미리보기를 제공합니다. HTML로 변환·복사·다운로드까지 지원합니다.

마크다운 입력 0줄 · 0자
미리보기 -
HTML 코드 

    

  


  
  

    
마크다운 변환기 사용 방법

    
왼쪽 편집창에 마크다운을 입력하면 오른쪽에 실시간으로 HTML 미리보기가 표시됩니다. 툴바 버튼으로 빠르게 마크다운 문법을 삽입할 수 있습니다.


    
마크다운 주요 문법

    

      # 제목1 / ## 제목2 / ### 제목3

      **굵게** / *기울임* / ~~취소선~~

      `인라인 코드` / ```코드블록```

      - 목록 / 1. 번호 목록

      > 인용문 / --- 구분선

      [링크텍스트](URL) / ![이미지](URL)

      | 표 | 헤더 | / |---|---| / | 내용 |
    


    
마크다운이 유용한 경우

    
GitHub README 작성, 블로그 포스팅, 노션·옵시디언 등 마크다운 기반 도구 사용, 기술 문서 작성, 이메일 초안 작성 등에 활용됩니다.

자주 묻는 질문

    

      
마크다운이란?

      
마크다운은 텍스트 기반의 간단한 마크업 언어입니다. 일반 텍스트로 서식 있는 문서를 작성할 수 있으며 GitHub, 노션, 블로그 등에서 널리 사용됩니다.

    

    

      
입력한 내용이 저장되나요?

      
아니요. 브라우저에서만 동작하며 서버에 저장되지 않습니다. 저장이 필요하면 .md 파일로 다운로드하세요.

    

    

      
표(테이블)는 어떻게 만드나요?

      
툴바의 '표' 버튼을 누르거나 | 헤더1 | 헤더2 | → |---|---| → | 내용1 | 내용2 | 형식으로 작성하세요.

    

    

      
HTML로 변환한 결과를 어떻게 사용하나요?

      
HTML 코드 보기 버튼을 눌러 HTML을 복사하거나 .html 파일로 저장해 웹페이지에 삽입할 수 있습니다.

    

    

      
코드 블록에 언어 지정이 되나요?

      
```javascript, ```python 등으로 언어를 지정할 수 있습니다. 현재 미리보기에서는 언어별 색상 강조는 지원되지 않습니다.
